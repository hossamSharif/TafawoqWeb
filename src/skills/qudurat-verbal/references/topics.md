## Reading Comprehension

### مواصفات النص
- **الطول**: 100-200 كلمة
- **اللغة**: عربية فصحى سليمة
- **المصادر**: علمية، أدبية، اجتماعية، تاريخية
- **التعقيد**: متدرج حسب الصعوبة

### أنواع الأسئلة

#### main-idea (الفكرة الرئيسية)
- ما الفكرة الرئيسية للنص؟
- ما الموضوع الذي يتناوله النص؟
- يدور النص حول...

#### inference (الاستنتاج)
- يُستنتج من النص أن...
- يُفهم ضمنياً من النص...
- ماذا يمكن أن نستنتج؟

#### detail (التفاصيل المباشرة)
- حسب النص، ...
- ذُكر في النص أن...
- أي العبارات صحيحة وفق النص؟

#### vocabulary-in-context (معنى كلمة في السياق)
- معنى كلمة "..." في النص:
- المقصود بـ "..." في الفقرة:

#### suitable-title (العنوان المناسب)
- أنسب عنوان للنص:
- العنوان الذي يعبر عن النص:

#### author-purpose (غرض الكاتب)
- هدف الكاتب من النص:
- يسعى الكاتب إلى...

  "passage": "النص الكامل..." // فقط في أول سؤال
}

## Sentence Completion

### single-blank (فراغ واحد)
```
الصدق من أهم _____ التي يجب أن يتحلى بها الإنسان.
(الصفات - الأفعال - الأقوال - الأماكن)
```

### double-blank (فراغين)
```
العلم _____ والجهل _____.
(نور/ظلام - سريع/بطيء - كبير/صغير - قديم/جديد)
```

### contrast (التضاد)
**مؤشرات**: لكن، بينما، على الرغم، إلا أن، غير أن
```
كان الجو حاراً، لكن الرياح كانت _____.
(باردة - قوية - هادئة - جافة)
```

### cause-effect (السبب والنتيجة)
**مؤشرات**: لأن، لذلك، بسبب، نتيجة، مما أدى
```
نجح الطالب لأنه _____.
(اجتهد - نام - لعب - غاب)
```

## Context Error

### semantic-error (خطأ دلالي)
كلمة لا تناسب سياق الجملة معنوياً.
```
يتميز الأسد بالجُبن في مواجهة أعدائه.
الخطأ: الجُبن (الصواب: الشجاعة)
```

### contradiction (التناقض)
عبارتان متناقضتان في نفس الجملة.
```
ذهب أحمد إلى المدرسة وبقي في البيت.
الخطأ: بقي في البيت
```

### طريقة السؤال
```
"حدد الكلمة الخطأ في الجملة التالية:"
أو
"الكلمة التي لا تتناسب مع السياق هي:"
```

## Odd Word

### category-based (الفئة)
كلمة لا تنتمي لنفس الفئة.
```
(تفاح - برتقال - جزر - عنب)
الشاذ: جزر (خضار، الباقي فواكه)
```

### semantic-based (المعنى)
كلمة مختلفة في المعنى.
```
(سعيد - مبتهج - فرح - حزين)
الشاذ: حزين (عكس الباقي)
```

### function-based (الوظيفة)
كلمة مختلفة في الوظيفة.
```
(قلم - ممحاة - كتاب - مسطرة)
الشاذ: كتاب (ليس أداة كتابة)
```
