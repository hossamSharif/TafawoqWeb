# Analogy Relations Reference

## الـ 22 علاقة المدعومة في التناظر اللفظي

### 1. synonymy (الترادف)
الكلمتان لهما نفس المعنى أو قريب منه.
```
سعيد : مبتهج    |    كريم : سخي    |    شجاع : جريء
```

### 2. antonymy (التضاد)
الكلمتان متعاكستان في المعنى.
```
علم : جهل    |    نور : ظلام    |    حر : برد
```

### 3. part-whole (جزء من كل)
الأولى جزء من الثانية.
```
صفحة : كتاب    |    إصبع : يد    |    غرفة : بيت
```

### 4. whole-part (كل إلى جزء)
الأولى كل والثانية جزء منها.
```
شجرة : ورقة    |    سيارة : عجلة    |    جسم : عضو
```

### 5. cause-effect (السبب والنتيجة)
الأولى سبب للثانية.
```
نار : دخان    |    مرض : ألم    |    اجتهاد : نجاح
```

### 6. effect-cause (النتيجة والسبب)
الأولى نتيجة للثانية.
```
شفاء : دواء    |    ظل : شمس    |    ثمرة : زراعة
```

### 7. tool-user (الأداة والمستخدم)
الأولى أداة يستخدمها الثاني.
```
مبضع : جراح    |    فرشاة : رسام    |    مطرقة : نجار
```

### 8. profession-tool (المهنة والأداة)
الأول صاحب مهنة والثانية أداته.
```
طبيب : سماعة    |    مزارع : منجل    |    خياط : إبرة
```

### 9. profession-action (المهنة والفعل)
الأول صاحب مهنة والثاني فعله الرئيسي.
```
معلم : تدريس    |    طبيب : علاج    |    قاضٍ : حكم
```

### 10. place (العلاقة المكانية)
الأول يوجد في الثاني أو يعمل فيه.
```
طالب : مدرسة    |    مريض : مستشفى    |    سجين : سجن
```

### 11. temporal / succession (التعاقب الزمني)
الأولى تسبق أو تعقب الثانية زمنياً.
```
فجر : شروق    |    ربيع : صيف    |    طفولة : شباب
```

### 12. degree / gradation (الدرجة والشدة)
الثانية أقوى أو أضعف من الأولى.
```
غضب : هياج    |    حب : عشق    |    مشي : ركض
```

### 13. type-of / category (النوع والصنف)
الأولى نوع من الثانية.
```
تفاح : فاكهة    |    أسد : حيوان    |    قمح : حبوب
```

### 14. made-of (المادة الخام)
الأولى مصنوعة من الثانية.
```
خبز : قمح    |    ورق : شجر    |    زجاج : رمل
```

### 15. attribute (الصفة)
الثانية صفة ملازمة للأولى.
```
سكر : حلاوة    |    ثلج : برودة    |    نار : حرارة
```

### 16. origin-branch (الأصل والفرع)
الثانية متفرعة أو مشتقة من الأولى.
```
لغة : لهجة    |    دين : مذهب    |    علم : تخصص
```

### 17. function (الوظيفة)
الثانية وظيفة الأولى.
```
قلب : ضخ    |    عين : إبصار    |    أذن : سمع
```

### 18. container (الوعاء والمحتوى)
الأولى وعاء للثانية.
```
كأس : ماء    |    قلم : حبر    |    خزانة : ملابس
```

### 19. condition (الشرط والنتيجة)
الأولى شرط لتحقق الثانية.
```
بذرة : شجرة    |    تعلم : معرفة    |    مال : شراء
```

### 20. transformation (التحول)
الأولى تتحول إلى الثانية.
```
يرقة : فراشة    |    ماء : ثلج    |    عجين : خبز
```

### 21. conjugation (التصريف اللغوي)
علاقة صرفية بين الكلمتين.
```
كتب : كاتب    |    علم : عالم    |    قرأ : قارئ
```

### 22. spatial (العلاقة الاتجاهية)
علاقة مكانية اتجاهية.
```
شرق : غرب    |    فوق : تحت    |    يمين : شمال
```

---

## قواعد تحديد العلاقة

1. **اقرأ الزوج الأصلي** وحدد العلاقة بدقة
2. **اختبر كل خيار** بنفس العلاقة
3. **الترتيب مهم** - (أداة : مستخدم) ≠ (مستخدم : أداة)
4. **تجنب العلاقات السطحية** - ابحث عن العلاقة الجوهرية
