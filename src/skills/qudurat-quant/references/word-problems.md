# Word Problem Templates - Quantitative Reasoning

## Purpose
Templates and guidelines for generating realistic word problems in five categories for GAT quantitative reasoning. All problems must use culturally appropriate contexts, realistic Arabic names, and mental-math-friendly numbers.

---

## 1. Speed-Time-Distance Problems (السرعة والمسافة والزمن)

### Core Formulas
```
المسافة = السرعة × الزمن
السرعة = المسافة ÷ الزمن
الزمن = المسافة ÷ السرعة
```

### Common Scenarios
1. **Travel between cities** (السفر بين المدن)
2. **Relative speed** (السرعة النسبية)
3. **Meeting problems** (مسائل اللقاء)
4. **Catching up problems** (مسائل اللحاق)

### Mental-Math-Friendly Values
- **Speeds**: 40, 50, 60, 80, 100, 120 km/h
- **Distances**: 100, 120, 150, 200, 240, 300, 400 km
- **Times**: 1, 1.5, 2, 2.5, 3, 4, 5 hours

### Realistic Arabic Names
**Male**: أحمد، محمد، خالد، عبدالله، سعود، فهد، ناصر، علي، عمر، يوسف
**Female**: فاطمة، نورة، سارة، مريم، عائشة، هدى، ريم، منى، لمى، شهد

### Saudi Cities (for context)
الرياض، جدة، مكة المكرمة، المدينة المنورة، الدمام، الطائف، تبوك، أبها، الخبر، ينبع

### Template Examples

#### Template 1: Simple Travel
```
انطلق [Name] من [City1] متجهاً إلى [City2] بسرعة [Speed] كم/ساعة.
إذا قطع مسافة [Distance] كم، فكم ساعة استغرقت الرحلة؟
```

#### Template 2: Relative Speed (Meeting)
```
خرج [Name1] من [City1] و[Name2] من [City2] في نفس الوقت متجهين للقاء بعضهما.
إذا كانت سرعة [Name1] هي [Speed1] كم/ساعة، وسرعة [Name2] هي [Speed2] كم/ساعة،
والمسافة بين المدينتين [Distance] كم، فبعد كم ساعة يلتقيان؟
```

#### Template 3: Catching Up
```
غادر [Name1] من [City1] بسرعة [Speed1] كم/ساعة. بعد [Time] ساعة،
تبعه [Name2] من نفس المكان بسرعة [Speed2] كم/ساعة.
بعد كم ساعة من انطلاق [Name2] سيلحق ب[Name1]؟
```

## 2. Work Problems (مسائل العمل)

### Core Formulas
```
العمل = المعدل × الزمن
إذا كان A ينجز العمل في x أيام، فمعدله = 1/x من العمل في اليوم
العمل المشترك = معدل A + معدل B
```

### Common Scenarios
1. **Individual completion** (إنجاز فردي)
2. **Joint work** (عمل مشترك)
3. **Partial work then joint** (عمل جزئي ثم مشترك)
4. **Pipes and tanks** (صنابير وخزانات)

### Mental-Math-Friendly Values
- **Days to complete alone**: 2, 3, 4, 5, 6, 8, 10, 12, 15, 20, 24, 30
- **Work rates**: 1/2, 1/3, 1/4, 1/5, 1/6, 1/8, 1/10, 1/12

### Occupations (for context)
عامل، نجار، سباك، كهربائي، رسام، بناء، خياط، مزارع

### Template Examples

#### Template 1: Joint Work
```
يستطيع [Name1] إنجاز عمل معين في [Days1] أيام، بينما يستطيع [Name2] إنجاز نفس العمل في [Days2] أيام.
إذا عملا معاً، فكم يوماً يحتاجان لإنجاز العمل؟
```

#### Template 2: Pipes Filling Tank
```
يمكن لصنبور ملء خزان في [Time1] ساعات، بينما يمكن لصنبور آخر ملء نفس الخزان في [Time2] ساعات.
إذا فُتح الصنبوران معاً، فكم ساعة يستغرق ملء الخزان؟
```

#### Template 3: Partial Then Joint
```
بدأ [Name1] عملاً يستطيع إنجازه وحده في [TotalDays] أيام. بعد [PartialDays] أيام،
انضم إليه [Name2] الذي يستطيع إنجاز نفس العمل وحده في [Days2] أيام.
كم يوماً احتاجا معاً لإنجاز بقية العمل؟
```

## 3. Age Problems (مسائل الأعمار)

### Core Formulas
```
العمر الحالي = العمر في الماضي + عدد السنوات
العمر في المستقبل = العمر الحالي + عدد السنوات
مجموع الأعمار = عمر A + عمر B + ...
```

### Common Scenarios
1. **Age comparison** (مقارنة الأعمار)
2. **Age relationships** (علاقات الأعمار)
3. **Past, present, future ages** (الأعمار في أزمنة مختلفة)
4. **Age ratios** (نسب الأعمار)

### Mental-Math-Friendly Values
- **Current ages**: 5, 8, 10, 12, 15, 18, 20, 24, 25, 30, 32, 36, 40, 45, 48, 50, 60
- **Years ago/from now**: 2, 3, 4, 5, 8, 10, 12, 15, 20
- **Ratios**: 2:1, 3:1, 3:2, 4:1, 5:2, 2:3, 3:4

### Family Relationships (for context)
أب، أم، ابن، ابنة، أخ، أخت، جد، جدة، عم، عمة، خال، خالة

### Template Examples

#### Template 1: Age Ratio
```
عمر [Person1] الآن يساوي [Ratio] أمثال عمر [Person2].
إذا كان مجموع عمريهما [Sum] سنة، فما عمر [Person1]؟
```

#### Template 2: Age Difference Over Time
```
عمر [Parent] الآن [Age1] سنة، وعمر [Child] الآن [Age2] سنة.
بعد كم سنة سيكون عمر [Parent] ضعف عمر [Child]؟
```

#### Template 3: Past Age Relationship
```
قبل [Years] سنوات، كان عمر [Person1] يساوي [Ratio] أمثال عمر [Person2].
إذا كان عمر [Person1] الآن [CurrentAge] سنة، فما عمر [Person2] الآن؟
```

## 4. Profit and Loss (الربح والخسارة)

### Core Formulas
```
الربح = سعر البيع - سعر الشراء (التكلفة)
نسبة الربح = (الربح ÷ التكلفة) × 100%
سعر البيع = التكلفة + الربح = التكلفة × (1 + نسبة الربح)
الخسارة = التكلفة - سعر البيع
نسبة الخسارة = (الخسارة ÷ التكلفة) × 100%
```

### Common Scenarios
1. **Simple profit/loss** (ربح أو خسارة بسيطة)
2. **Percentage profit/loss** (نسبة الربح أو الخسارة)
3. **Successive discounts** (خصومات متتالية)
4. **Marked price and selling price** (السعر المعلن وسعر البيع)

### Mental-Math-Friendly Values
- **Cost prices**: 100, 120, 150, 200, 240, 300, 400, 500, 600, 800, 1000 ريال
- **Profit percentages**: 5%, 10%, 15%, 20%, 25%, 30%, 40%, 50%
- **Loss percentages**: 5%, 10%, 15%, 20%, 25%
- **Discounts**: 10%, 15%, 20%, 25%, 30%, 40%, 50%

### Products (for context)
كتاب، هاتف، ساعة، حقيبة، جهاز، سلعة، بضاعة، منتج

### Template Examples

#### Template 1: Simple Profit Percentage
```
اشترى [Name] [Product] بمبلغ [Cost] ريال، ثم باعه بربح [Profit%]%.
ما سعر بيع [Product]؟
```

#### Template 2: Find Cost Given Selling Price and Profit
```
باع [Name] [Product] بمبلغ [SellingPrice] ريال محققاً ربحاً قدره [Profit%]%.
ما ثمن شراء [Product]؟
```

#### Template 3: Successive Discounts
```
كان سعر [Product] المعلن [MarkedPrice] ريال. خُفّض السعر بنسبة [Discount1]%،
ثم خُفّض مرة أخرى بنسبة [Discount2]% من السعر الجديد.
ما السعر النهائي للمنتج؟
```

#### Template 4: Break-Even Analysis
```
اشترى [Name] [Quantity] [Product] بسعر [UnitCost] ريال للواحد.
باع [SoldQuantity] منها بسعر [SellingPrice] ريال للواحد.
بكم يجب أن يبيع الباقي ليحقق ربحاً إجمالياً قدره [TotalProfit] ريال؟
```

## 5. Mixture Problems (مسائل الخلط والمزج)

### Core Formulas
```
إجمالي الكمية = كمية A + كمية B
إجمالي القيمة = (كمية A × سعر A) + (كمية B × سعر B)
متوسط السعر = إجمالي القيمة ÷ إجمالي الكمية
إجمالي التركيز = (كمية A × تركيز A) + (كمية B × تركيز B)
```

### Common Scenarios
1. **Mixing liquids with different concentrations** (خلط سوائل بتركيزات مختلفة)
2. **Mixing items with different prices** (خلط أصناف بأسعار مختلفة)
3. **Dilution problems** (مسائل التخفيف)
4. **Alligation (mixing to get desired average)** (الخلط للحصول على متوسط معين)

### Mental-Math-Friendly Values
- **Quantities**: 2, 3, 4, 5, 6, 8, 10, 12, 15, 20, 24, 30 kg or liters
- **Concentrations**: 5%, 10%, 15%, 20%, 25%, 30%, 40%, 50%, 60%, 75%, 80%, 100%
- **Prices per kg**: 10, 12, 15, 20, 24, 25, 30, 40, 50 ريال

### Products (for context)
قهوة، شاي، أرز، سكر، عسل، ملح، دقيق، زيت، عصير، حليب، خل، كحول (طبي)

### Template Examples

#### Template 1: Mixing Two Types by Price
```
خلط [Name] [Quantity1] كيلوغرام من [Product1] سعر الكيلو منه [Price1] ريال
مع [Quantity2] كيلوغرام من [Product2] سعر الكيلو منه [Price2] ريال.
ما متوسط سعر الكيلوغرام الواحد من الخليط؟
```

#### Template 2: Mixing to Get Desired Average
```
يريد [Name] خلط نوعين من [Product]، الأول سعر الكيلو منه [Price1] ريال،
والثاني سعر الكيلو منه [Price2] ريال، للحصول على خليط سعر الكيلو منه [TargetPrice] ريال.
ما النسبة التي يجب خلط النوعين بها؟
```

#### Template 3: Concentration Mixture
```
خُلطت [Quantity1] لتر من محلول تركيزه [Concentration1]% مع [Quantity2] لتر من محلول تركيزه [Concentration2]%.
ما تركيز المحلول الناتج؟
```

#### Template 4: Dilution
```
كم لتراً من الماء يجب إضافته إلى [Quantity] لتر من محلول تركيزه [InitialConc]%
للحصول على محلول تركيزه [FinalConc]%؟
```

## Generation Guidelines

### Cultural Appropriateness
1. **Use realistic Saudi/Gulf Arabic names** from the provided lists
2. **Use familiar contexts**: travel between Saudi cities, common occupations, everyday products
3. **Avoid**:
   - Interest/riba (الربا) in financial problems - use profit/loss instead
   - Alcohol in mixture problems - use medical alcohol (كحول طبي) or other substances
   - Culturally inappropriate scenarios

### Number Selection Rules
1. **Choose numbers that allow mental calculation**:
   - Prefer whole numbers and common fractions (1/2, 1/3, 1/4, 1/5)
   - Avoid decimals with more than 1 decimal place
   - Use numbers that divide evenly when possible
2. **Realistic ranges**:
   - Ages: 5-80 years
   - Speeds: 40-120 km/h for cars, 500-900 km/h for planes
   - Money: 10-10,000 ريال for consumer products
   - Distances: 50-1000 km between cities

### Distractor Generation (Wrong Answer Choices)
For each correct answer, generate 3 plausible distractors based on common errors:

1. **Formula Error**: Using wrong formula (e.g., adding instead of multiplying)
2. **Calculation Error**: Arithmetic mistake (e.g., 120/3 = 30 instead of 40)
3. **Unit Error**: Forgetting to convert units or partial calculation
4. **Conceptual Error**: Misunderstanding the problem setup

### Example Distractor Pattern (Speed Problem)
```
Correct: Distance = Speed × Time = 60 × 3 = 180 km

Distractors:
- 240 km (using 60 × 4 - adding an extra hour)
- 20 km (dividing instead of multiplying: 60 ÷ 3)
- 63 km (calculation error: 60 + 3)
```

## Step-by-Step Solution Format

Every word problem solution MUST include:

### 1. المعطيات (Given Information)
List all known values with clear Arabic labels

### 2. المطلوب (What is Required)
Clearly state what needs to be found

### 3. الحل (Solution Steps)
**Step 1**: Identify the formula or approach
**Step 2**: Substitute values
**Step 3**: Calculate
**Step 4**: Verify units and reasonableness

### 4. الإجابة النهائية (Final Answer)
State the answer with proper units

### Example Solution Template
```markdown
**المعطيات:**
- السرعة = 60 كم/ساعة
- الزمن = 3 ساعات

**المطلوب:** إيجاد المسافة المقطوعة

**الحل:**
**الخطوة 1:** نستخدم القانون: المسافة = السرعة × الزمن

**الخطوة 2:** نعوض القيم:
المسافة = 60 × 3

**الخطوة 3:** نحسب:
المسافة = 180 كم

**الخطوة 4:** نتحقق: السرعة معقولة، والزمن معقول، إذاً المسافة منطقية

**الإجابة النهائية:** 180 كم
```

## Quality Checklist

Before finalizing any word problem, verify:
- [ ] Uses realistic Arabic name(s) from provided lists
- [ ] Context is culturally appropriate
- [ ] Numbers allow mental calculation (no complex decimals)
- [ ] Answer is a whole number or simple fraction
- [ ] All 4 answer choices are plausible
- [ ] Step-by-step solution is complete and clear in Arabic
- [ ] Solution uses proper Arabic mathematical terminology
- [ ] Units are consistent and clearly stated
- [ ] Problem statement is unambiguous
- [ ] Difficulty level is appropriate for GAT (high school level)
