## Arithmetic

### basic-operations
- الجمع، الطرح، الضرب، القسمة
- ترتيب العمليات: أقواس → أسس → ضرب/قسمة → جمع/طرح
- خصائص: التبديل، التجميع، التوزيع

### number-properties
- **أولية**: 2, 3, 5, 7, 11, 13, 17, 19, 23...
- **زوجية/فردية**: زوجي×فردي=زوجي، فردي+فردي=زوجي
- **قابلية القسمة**:
  - ÷2: آحاد زوجي
  - ÷3: مجموع الأرقام ÷3
  - ÷4: آخر رقمين ÷4
  - ÷5: آحاد 0 أو 5
  - ÷9: مجموع الأرقام ÷9
- **ق.م.أ**: أكبر عدد يقسم العددين
- **م.م.أ**: أصغر عدد يقبل القسمة على العددين

### fractions
- تبسيط: قسمة البسط والمقام على ق.م.أ
- توحيد المقامات: م.م.أ للمقامات
- ضرب: بسط×بسط / مقام×مقام
- قسمة: ضرب بمقلوب الثاني
- مقارنة: توحيد المقامات أو الضرب التبادلي

### decimals
- تحويل كسر→عشري: قسمة البسط على المقام
- تقريب: النظر للرقم التالي (<5 حذف، ≥5 جبر)
- دورية: 1/3=0.333..., 1/6=0.1666...

### exponents-roots
**قوانين الأسس:**
- aⁿ × aᵐ = aⁿ⁺ᵐ
- aⁿ ÷ aᵐ = aⁿ⁻ᵐ
- (aⁿ)ᵐ = aⁿˣᵐ
- a⁰ = 1
- a⁻ⁿ = 1/aⁿ
- a^(1/n) = ⁿ√a

**الجذور:**
- √(a×b) = √a × √b
- √(a/b) = √a / √b
- √a² = |a|

### ratio-proportion
- النسبة a:b = a/b
- التناسب الطردي: y = kx
- التناسب العكسي: y = k/x
- إذا a:b = c:d فإن ad = bc

### percentages
- النسبة = (الجزء/الكل) × 100
- الجزء = (النسبة × الكل) / 100
- زيادة x% ثم نقصان x% ≠ الأصل
- زيادة متتالية: (1+r₁)(1+r₂)

## Algebra

### linear-equations
- ax + b = c → x = (c-b)/a
- نظام معادلتين: الحذف أو التعويض

### quadratic-equations
- ax² + bx + c = 0
- التحليل: إيجاد عددين حاصل ضربهما c وجمعهما b
- القانون: x = (-b ± √(b²-4ac)) / 2a
- مجموع الجذرين = -b/a
- حاصل ضرب الجذرين = c/a

### inequalities
- الضرب/القسمة بسالب يعكس الإشارة
- x > a و x < b → a < x < b

### algebraic-expressions
- الفرق بين مربعين: a² - b² = (a+b)(a-b)
- المربع الكامل: a² ± 2ab + b² = (a±b)²
- مكعب مجموع: a³ + b³ = (a+b)(a²-ab+b²)

### sequences
**حسابية:** aₙ = a₁ + (n-1)d, Sₙ = n(a₁+aₙ)/2
**هندسية:** aₙ = a₁ × rⁿ⁻¹, Sₙ = a₁(1-rⁿ)/(1-r)

### functions
- f(x) = تعبير جبري
- مجال: قيم x الممكنة
- مدى: قيم f(x) الناتجة

## Geometry

### angles
- حادة: 0° < θ < 90°
- قائمة: θ = 90°
- منفرجة: 90° < θ < 180°
- متتامتان: مجموعهما 90°
- متكاملتان: مجموعهما 180°
- متقابلتان بالرأس: متساويتان

### triangles
- مجموع الزوايا = 180°
- **فيثاغورس**: a² + b² = c² (للقائم)
- **30-60-90**: النسب 1 : √3 : 2
- **45-45-90**: النسب 1 : 1 : √2
- المساحة = ½ × القاعدة × الارتفاع
- المساحة = ½ × a × b × sin(C)

### circles
- المحيط = 2πr = πd
- المساحة = πr²
- طول القوس = (θ/360) × 2πr
- مساحة القطاع = (θ/360) × πr²
- المماس ⊥ نصف القطر
- الزاوية المحيطية = ½ الزاوية المركزية

### polygons
| الشكل | المساحة | المحيط |
|-------|---------|--------|
| مربع | s² | 4s |
| مستطيل | l×w | 2(l+w) |
| معين | ½×d₁×d₂ | 4s |
| متوازي أضلاع | b×h | 2(a+b) |
| شبه منحرف | ½(a+b)×h | مجموع الأضلاع |

### 3d-shapes
| الشكل | الحجم | المساحة الكلية |
|-------|-------|---------------|
| مكعب | s³ | 6s² |
| متوازي مستطيلات | l×w×h | 2(lw+lh+wh) |
| أسطوانة | πr²h | 2πr(r+h) |
| مخروط | ⅓πr²h | πr(r+l) |
| كرة | ⁴⁄₃πr³ | 4πr² |

### coordinate-geometry
- المسافة = √[(x₂-x₁)² + (y₂-y₁)²]
- المنتصف = ((x₁+x₂)/2, (y₁+y₂)/2)
- الميل = (y₂-y₁)/(x₂-x₁)

## Statistics

### central-tendency
- المتوسط = مجموع القيم / عددها
- الوسيط = القيمة الوسطى (مرتبة)
- المنوال = الأكثر تكراراً

### dispersion
- المدى = أكبر قيمة - أصغر قيمة
- الربيعيات: Q1, Q2 (الوسيط), Q3

### probability
- P(A) = عدد النتائج المطلوبة / عدد النتائج الممكنة
- P(A و B) = P(A) × P(B) [مستقلة]
- P(A أو B) = P(A) + P(B) - P(A و B)

### permutations
- التباديل: nPr = n!/(n-r)!
- التوافيق: nCr = n!/[r!(n-r)!]

## Word Problems

### speed-time-distance
- المسافة = السرعة × الزمن
- السرعة المتوسطة = المسافة الكلية / الزمن الكلي

### work-problems
- معدل العمل = 1/الزمن
- معدل مشترك = 1/t₁ + 1/t₂

### age-problems
- فرق الأعمار ثابت دائماً
- استخدم متغيرات للعمر الحالي

### profit-loss
- الربح = سعر البيع - سعر الشراء
- نسبة الربح = (الربح/سعر الشراء) × 100

### mixture-problems
- C₁V₁ + C₂V₂ = C₃(V₁+V₂)
